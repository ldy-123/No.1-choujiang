# 抽奖小工具（本地运行）

一个简单、美观的抽奖网页工具，支持：

- **导入名单**：上传 Excel（`.xlsx/.xls/.csv`）或直接粘贴姓名（每行一个）
- **奖励设置**：奖励名称、数量、图片（可选）
- **发起抽奖**：按奖励顺序抽取，自动去重、不重复中奖
- **导出结果**：导出 CSV
- **本地保存/恢复**：保存到浏览器本地存储

## 如何运行

本项目不依赖 Node。推荐用本地服务器打开（避免浏览器对 `file://` 的限制）。

在本目录执行：

```bash
python3 -m http.server 5173
```

然后浏览器访问：

- `http://localhost:5173`

入口页面是 `index.html`。

## Excel 导入规则

- 默认读取**第一个工作表**
- 优先读取表头为 **“姓名”** 或 **“Name”** 的列（不区分大小写）
- 如果没有找到上述表头，则读取**第一列**
